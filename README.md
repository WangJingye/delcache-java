# website

####底层框架
    `spring + springMVC + jdbc`

####自动化代码生成工具
    `php -S 0.0.0.0:8081 -t  generate`

    技术菜暂时用php生成代码

####项目说明
    spring配置文件 src/main/webapp/WEB-INF/web.xml
    
    前端界面目录 src/main/webapp/html
    
    前端界面使用 sitemesh3 装饰，布局文件在src/main/webapp/layouts中
    
    静态资源文件目录 src/main/webapp/static
